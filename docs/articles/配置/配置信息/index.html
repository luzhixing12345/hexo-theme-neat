配置信息

如果您想修改zood的默认配置,你需要首先调出zood的配置文件

zood config

接下来您可以修改 md-docs/ 下生成的 _config.yml 文件

您可_config.yml的配置选项的命名及其注释自行修改

修改完成后如果您想永久保存修改可以将其重新注册回zood,注意此操作为替换操作,您所做的修改将成为全局配置文件并应用于之后所有的项目

zood -s

如果您想要支持其他语法高亮,请前往prismjs官网,选择您想要支持的语言,然后在其最下方点击下载js和css,到一个文件夹,并将这个文件夹将其移入md-docs后重命名为 prismjs

您的目录结构应该如下所示

- md-docs/
  - README.md
  - dir.yml
  - prismjs/
    - prism.css
    - prism.js

接着将其写回zood配置文件即可,注意此操作为替换操作,您所做的修改将成为全局配置文件并应用于之后所有的项目

zood -s
