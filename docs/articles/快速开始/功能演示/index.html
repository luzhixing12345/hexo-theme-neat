搜索

注:此功能需要修改config启用,如果希望启用此功能,请参考配置信息模块启用enable_search

键入 ctrl + k 试试看

搜索可以为您提供相关信息的搜索结果,注意搜索的范围是全局的,最多搜索结果8个

由于本文档为静态内容,所以如果文章内容很多搜索可能耗时较长,建议谨慎开启

此外您可在config文件中手动设置搜索的域,例如只在根目录和API目录下搜索(默认数组为空,代表全局)

# 是否启用查询搜索
enable_search: true

  search_scope: ['.','API']
